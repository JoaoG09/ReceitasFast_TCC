Sobre o Projeto

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) com o objetivo de facilitar a recomendação de receitas saudáveis com base nos ingredientes disponíveis e no tempo de preparo.

Com uma interface simples e intuitiva, o sistema permite ao usuário inserir os ingredientes que possui em casa, além do tempo que tem disponível, e retorna receitas adequadas às suas necessidades.

O projeto utiliza HTML, CSS e JavaScript puro para sua construção, mantendo o foco em acessibilidade, leveza e funcionalidade.
